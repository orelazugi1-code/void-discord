Install VOID

VOID installs alongside Vencord — requires Vencord to be installed first.

1
Install Vencord from vencord.dev if you haven't already.
2
Download VOID using the button below. It's a ZIP containing 4 files.
3
Extract the ZIP and copy the 4 files to %AppData%\Vencord\dist\ (replace existing).
4
Restart Discord completely (quit from tray, relaunch). Look for VOID in settings!
